# CopyTrackName - Аддон для копирования названия трека

Удобный скрипт для PulseSync, который добавляет иконку копирования рядом с названием трека.

## Описание

Скрипт автоматически добавляет иконку копирования рядом с названием каждого трека в интерфейсе PulseSync. При нажатии на иконку название трека копируется в буфер обмена в выбранном формате.

## Возможности

- ✅ Автоматическое добавление иконки копирования ко всем трекам
- ✅ Несколько форматов копирования:
  - Исполнитель - Название
  - Название - Исполнитель
  - Только исполнитель
  - Только название
  - Пользовательский формат
- ✅ Настройка внешнего вида иконки (цвет, размер, прозрачность)
- ✅ Визуальные уведомления (можно отключить)
- ✅ Работает как в плеере, так и в списках треков
- ✅ Автоматическое обнаружение новых треков на странице

## Использование

После активации скрипта рядом с названием каждого трека появится иконка копирования (📋). Просто нажмите на неё, для копирования информации о треке в буфер обмена.

## Настройки

### Основные настройки

- **Показывать иконку копирования** - включить/выключить отображение иконки
- **Формат копирования** - выбор формата из списка:
  - Исполнитель - Название
  - Название - Исполнитель
  - Только исполнитель
  - Только название
- **Показывать уведомления** - включить/выключить уведомления при копировании

### Внешний вид

- **Динамический цвет иконки** - автоматически определять цвет из темы страницы (по умолчанию включено)
- **Цвет иконки** - выбор цвета иконки (используется при отключенном динамическом цвете)
- **Размер иконки** - размер от 12 до 24 пикселей (по умолчанию 16)
- **Прозрачность иконки** - прозрачность в неактивном состоянии от 5 до 100% (по умолчанию 70%)

### Дополнительно

- **Пользовательский формат** - создайте свой формат используя переменные:
  - `{artist}` - имя исполнителя
  - `{title}` - название трека
  - Пример: `{artist} - {title}` или `♫ {title} by {artist}`
- **Использовать пользовательский формат** - включить использование пользовательского формата
